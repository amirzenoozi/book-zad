Politique de confidentialité

Dernière mise à jour : 2026-07-23

BookZad est conçu pour ne collecter aucune donnée personnelle. Il n'y a ni compte ni serveur.

Ce à quoi l'extension accède

  • Vos favoris — pour les afficher, les noter, les chercher et les organiser.
  • L'URL et le titre de l'onglet actif — pour la pastille sur l'icône et les actions rapides.
  • Le texte des pages que vous visitez — lu localement afin d'être comparé à vos favoris enregistrés.

Où vont les données

Nulle part. BookZad n'effectue aucune requête réseau vers un service externe.

  • Le texte des pages est découpé et comparé sur votre appareil avec un modèle TF-IDF local ; le texte lui-même n'est jamais envoyé.
  • Vos notes, étiquettes, scores et réglages résident dans le stockage d'extension du navigateur, sur votre appareil.
  • Les signaux d'usage (visites, fraîcheur, temps passé) sont suivis par l'extension elle-même — BookZad n'utilise pas la permission d'historique.

Supprimer vos données

Désinstaller l'extension supprime toutes les métadonnées stockées. Vos favoris natifs restent intacts.
